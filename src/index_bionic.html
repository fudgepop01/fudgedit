Hehe bionic text web component go brrrr

I saw this thing on twitter about 2 hours ago and figured:

Wow, it sure would be cool to have that applied to every HTML text node in a website automatically

It turns out, thanks to stenciljs, it's pretty easy

All I did was create a web component that goes through and finds each of the textnodes contained within it, breaks it up into chunks and surrounds the relevant chunks with <span> tags.

it goes and does this recursively too - so all I have is something like the following: 


        <to-bionic>
          omg I'm bionicified outside of a div
          <p>
            omg i'm bionicified inside of a div
          </p>
          <p no-bionic>
            no bionic text for me
          </p>
          yep, I still work as intended
        </to-bionic>

yes, within tags with a "no-bionic" attribute, this effect is disabled (like it is now)

scripts are also ignored - but yes, I'll put this out there a little later.
